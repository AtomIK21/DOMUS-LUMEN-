Domus Lumen

  • Este proyecto tiene como objetivo diseñar y gestionar tablas de enrutamiento para optimizar la transferencia de datos en una red. Las tablas de enrutamiento permiten determinar la mejor ruta para enviar los paquetes de datos de un origen a un destino, asegurando una comunicación eficiente y confiable.
